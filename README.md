# particles

Simple particle system example written in C99.

## Build

Clone with submodules

`git clone --recurse-submodules https://github.com/jparimaa/particles.git`

Configure and build, for example

`cmake . && make`

Run!

## Tools

As submodules
- cglm
- glfw
- stb

Other
- glad
